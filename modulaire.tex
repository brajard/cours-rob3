\begin{frame}
\frametitle{Etat des lieux}
\begin{itemize}
\setlength\itemsep{1.2em}
\item Vos programmes sont de plus en plus gros.
\item Certaines fonctionnalités ne sont pas spécifiques à un seul programme
(exemple fonctions d'E/S).
\item Le maintien et la compréhension des programmes est difficile.
\item le travail collaboratif est presque impossible.
\item La compilation de tout le projet est nécessaire à chaque modification (même minime).
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{Une solution : la programmation modulaire}

\begin{block}{}
Possibilité de répartir un programme
sur \red{plusieurs fichiers} réunissant les
fonctionnalités et définitions d'un aspect
particulier du programme.
\end{block}
Chacun de ces découpages est appelé 
\red{un module}.\\
Ex : 
\begin{itemize}
\item Un module d'E/S, 
\item Un module pour les calculs,
\item Un module contenant le \bvrb|main()|.
\end{itemize}
\vspace{1em}
$\rightarrow$ nécessite une réflexion sur le découpage de votre
code.
\end{frame}

\begin{frame}[fragile]
\frametitle{Que contient un module ?}
\begin{block}{}
Un module est composé :
\begin{itemize}
\item Un fichier entête \bvrb|.h|
\item Un fichier source \bvrb|.c|
\end{itemize}
\end{block}
\begin{itemize}
\item Le fichier entête décrit l'interface 
du module.
\item Le fichier source contient l'implémentation
(la définition) des fonctions.
\end{itemize}
\vspace{1em}
Le fichier entête est inclus par la directive :
\begin{codeblock}{}
\vspace{-.3cm}
\lstset{escapeinside={§§}}
\lstset{basicstyle=\scriptsize}
\begin{codeC}
#include "entete.h"
\end{codeC}
\vspace{-.3cm}
\end{codeblock}
\end{frame}

\begin{frame}[fragile]
\frametitle{Le fichier entête (header) .h}
Le fichier entête contient :
\begin{itemize}
\item Des directives \bvrb|#include|
\item Des directives \bvrb|#define|
\item Les prototypes des fonctions du module
utilisables par les autres modules
\item des \red{déclaration} de variables globales
\item Des modèles de structure.
\end{itemize}

\begin{alertblock}{}
Eviter de \red{définir} des variables (initialisation, etc.)
\end{alertblock}

\end{frame}
