\begin{frame}
\frametitle{Votre mission}
Vous avez reçu une machine de Turing cassée ainsi que le 
ruban que la machine avait écrite avant de se casser.\\
\vspace{2em}
Imaginez que cette machine est la clé du cryptage
d'une arme ultra-secrète, ou qu'elle permette de fournir
une source inépuisable d'énergie propre.
\vspace{2em}
\begin{alertblock}{}
Votre mission est de la réparer.
\end{alertblock}
\end{frame}

\begin{frame}
\frametitle{Comment faire ...}
\framesubtitle{...pour sauver le monde ?}

\begin{enumerate}
\item Connectez-vous à votre QG :\\
\url{http://jbrlod.locean-ipsl.upmc.fr/ROB/connection.php}
%\item Récupérez votre mot de passe à l'aide de votre numéro d'agent (numéro d'étudiant).
\item Entrez votre numéro d'agent (numéro d'étudiant) et votre mot de
  passe (= numéro d'étudiant).
\item Le site vous affiche la machine que vous devez réparer (chaque agent a une machine
différente mais de complexité équivalente).
\item Le compte à rebours est lancé : vous avez \red{1 heure} pour réparer la machine
\item Vous avez \red{2 essais}.
\item Si vous vous reconnectez trop tard ou soumettez une réparation trop tard, 
le site vous donne la réponse.
\end{enumerate}

\end{frame}

\begin{frame}
\frametitle{Attention !!!!}
\begin{alertblock}{}
Avant d'entreprendre la réparation d'une machine,
assurez-vous que vous avez une heure libre devant vous avec une connexion internet\\
Vous pouvez également aller au camp d'entrainement (accessible sur le site)
autant de fois que vous voulez.
\end{alertblock}
Vous avez jusqu'au 22 Octobre (inclus) pour remplir votre mission.
\end{frame}

\item Machine réparée au bout de 2 essais : 2.5 pts
\item Machine réparée en moins de 15 min : +1 pt
\end{itemize}
